Blocs de ma formation Capitaine WP pour apprendre à créer des blocs Gutenberg en JavaScript et React.

== Lancer le projet ==

La première fois :

`npm install`

À chaque fois :

`npm start`

Et pour construire la build de production :

`npm run build`
